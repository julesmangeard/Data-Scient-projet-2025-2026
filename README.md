# Data-Science-projet-2025-2026
Ce GitHub vise à présenter le projet en Data Science du groupe (Jules Mangeard -Margot Schmutz - Pierre Launay - Eliot Nehme) sur le thème suivant : Isolation-based Anomaly Detection

Il est au départ constitué d'un petit code R qui a servi à importer nos données. Cette base de données doit être nommée "data" pour le code Python.

Pour faciliter la tâches, nous avons déposé ces données directement sur GitHub en format xlsx (ok car ce n'est pas une grosse base de données)

Pour ensuite être consititué d'un code Python relatif à l'application de l'algorithme d'isolation sur ces données.
